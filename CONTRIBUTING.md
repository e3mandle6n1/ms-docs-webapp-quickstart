# Contributing

Thanks for your interest in improving this project.

## How To Contribute

1. Open an issue describing the bug or feature request.
2. Create a branch for your change.
3. Keep pull requests focused and small.
4. Add tests when behavior changes.
5. Update docs when setup or usage changes.

## Pull Request Checklist

- Clear description of what changed and why.
- Local run succeeds (`uvicorn main:app --reload`).
- Azure deployment config remains valid (`azd up` / `azd deploy`).
- No unrelated refactors mixed into the same PR.

## Reporting Issues

When filing a bug, include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Logs or stack traces (if available)

Thank you for contributing.
